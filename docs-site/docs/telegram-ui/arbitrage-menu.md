# 🎯 Меню арбитража

Меню выбора функций арбитража - основной интерфейс для доступа к автоматическому поиску торговых возможностей.

## Визуальное представление

```plaintext
┌─────────────────────────────────┐
│ Меню арбитража                  │
├─────────────────────────────────┤
│ [🔍 Сканировать] [🎮 Выбор игры]│
│ [📊 Уровни] [⚙️ Настройки]      │
│ [🤖 Авто-арбитраж]              │
│ [◀️ Назад]                      │
└─────────────────────────────────┘
```

## Кнопки меню

### 🔍 Сканировать

Быстрое сканирование рынка для поиска арбитражных возможностей.

**Действие**: Запускает сканирование с текущими настройками  
**Callback**: `arb_scan`

!!! tip "Совет"
    Рекомендуется сначала выбрать игру и уровень, затем нажать "Сканировать"

### 🎮 Выбор игры

Выбор игры для сканирования:

- 🎮 CS:GO/CS2
- ⚔️ Dota 2
- 🔧 TF2
- 🏗️ Rust

**Callback**: `arb_game`

### 📊 Уровни

Выбор уровня арбитража:

=== "🚀 Разгон"
    **Диапазон**: $0.50 - $3  
    **ROI**: 15-30%  
    **Риск**: Низкий
    
    Идеально для начинающих трейдеров с малым капиталом.

=== "📊 Стандарт"
    **Диапазон**: $3 - $10  
    **ROI**: 10-20%  
    **Риск**: Средний
    
    Оптимальный баланс между риском и прибылью.

=== "💼 Средний"
    **Диапазон**: $10 - $30  
    **ROI**: 7-15%  
    **Риск**: Средний
    
    Для опытных трейдеров с пониманием рынка.

=== "🏆 Продвинутый"
    **Диапазон**: $30 - $100  
    **ROI**: 5-10%  
    **Риск**: Высокий
    
    Высокие ставки, требуется терпение для продажи.

=== "💎 Профессионал"
    **Диапазон**: $100+  
    **ROI**: 3-7%  
    **Риск**: Очень высокий
    
    Только для профессионалов с большим капиталом.

### ⚙️ Настройки

Настройка параметров сканирования:

- Минимальная прибыль (%)
- Максимальная цена
- Фильтры по качеству
- Лимиты на количество предметов

### 🤖 Авто-арбитраж

Автоматическое сканирование и создание таргетов при находке возможностей.

!!! warning "Внимание"
    Авто-арбитраж создает таргеты автоматически. Убедитесь, что настроены правильные лимиты.

## Расширенное меню

Расширенное меню включает дополнительные функции:

```plaintext
┌─────────────────────────────────────┐
│ Расширенное меню арбитража          │
├─────────────────────────────────────┤
│ [⚡ Упрощенное меню]                │
│ [🔎 ВСЕ СТРАТЕГИИ]                  │
│ [🚀 Быстрый скан] [🔬 Глубокий скан]│
│ [📈 Анализ рынка] [🔍 Многоуровневый]│
│ [🎯 Float арбитраж] [📝 Расширенные]│
│ [⚡ Enhanced Scanner] [📊 Статистика]│
│ [🎯 Создать таргет] [🔄 Сравнить]  │
│ [💎 Waxpeer P2P]                    │
│ [◀️ Главное меню]                   │
└─────────────────────────────────────┘
```

### Дополнительные функции

**🚀 Быстрый скан**: Сканирование топ-100 предметов  
**🔬 Глубокий скан**: Полное сканирование всех предметов  
**📈 Анализ рынка**: Технический анализ и тренды  
**🎯 Float арбитраж**: Арбитраж на основе float value  
**📝 Расширенные ордера**: Сложные торговые стратегии  
**💎 Waxpeer P2P**: Кросс-платформенный арбитраж
